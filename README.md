# Mini Shop

SPA-приложение интернет-магазина на Vue 3 + TypeScript с интеграцией DummyJSON API.

## Стек

- **Vue 3** — Composition API, `<script setup lang="ts">`
- **TypeScript**
- **Vue Router 4** — маршрутизация, lazy-loading, catch-all 404
- **Pinia** — управление состоянием
- **Axios** — HTTP-клиент
- **Lucide** — иконки
- **Vitest** — unit-тесты
- **Vite** — сборка

## Функционал

- Каталог с фильтрацией по категориям, поиском (debounce), сортировкой и фильтром по цене
- Страница товара с рейтингом, описанием и кнопками действий
- Корзина с изменением количества, итоговой суммой и переходом к оформлению
- Оформление заказа с валидацией формы
- Избранное (хранение по id, данные из кэша каталога)
- Toast-уведомления при добавлении в корзину / избранное
- Skeleton-лоадеры при загрузке каталога
- Анимация переходов между страницами
- Страница 404
- Сохранение корзины и избранного в localStorage
- Адаптивная вёрстка (мобильная / планшет / десктоп)

## Запуск

```bash
npm install
npm run dev
```

## Тесты

